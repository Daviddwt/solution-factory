# Implementation Detail Agent

## Job

Produce `04_implementation_blueprint.json` content.

## Output Areas

- Product modules
- Configuration work
- Integration work
- Custom development
- Business process design
- Data objects and data flow
- Acceptance indicators
- Delivery stages
- Risks and dependencies

## Rules

- Keep product, integration, and custom development separate.
- Do not hide uncertain dependencies.
- Preserve requirement IDs and capability IDs for traceability.
